# busyatra-bus-booking
A complete bus ticket booking system for Nepal built with Node.js, Express, MongoDB. Features user authentication, bus search with Nepali city autocomplete, interactive seat selection, online payments, booking management, and admin panel. Perfect for travel agencies and bus operators.

A comprehensive bus ticket booking platform for Nepal with modern UI, real-time seat selection, and secure payments.


## ✨ Features

### 👤 User Features
- **User Authentication**: Register/Login with JWT
- **Bus Search**: Search by source, destination, date
- **Nepali Cities**: Autocomplete for 50+ Nepali cities
- **Interactive Seat Selection**: Visual seat layout with real-time pricing
- **Multiple Payments**: eSewa, Khalti, Credit Card, Net Banking
- **Booking Management**: View/cancel bookings, download tickets
- **Email Confirmation**: Automatic email after booking

### 🛠️ Admin Features
- **Dashboard**: Real-time statistics and analytics
- **Bus Management**: Add/edit/delete buses
- **Route Management**: Manage routes and fares
- **Schedule Management**: Create/update bus schedules
- **Booking Overview**: View all bookings with filters

### 🎨 UI/UX
- Responsive design (mobile/tablet/desktop)
- Modern gradient themes
- Smooth animations
- Loading states and error handling
- Nepali language support
